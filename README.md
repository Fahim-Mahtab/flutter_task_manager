# Flutter Task Manager

A simple and intuitive task management app built with Flutter. Organize, prioritize, and keep track of your daily tasks efficiently!

## Features

- Add, edit, and delete tasks
- Mark tasks as completed or pending
- Organize tasks with categories or priorities
- User-friendly and responsive UI
- Cross-platform: Works on Android, iOS, and the web

## Getting Started

### Prerequisites

- [Flutter](https://flutter.dev/docs/get-started/install) SDK installed
- Compatible IDE: VS Code, Android Studio, or IntelliJ IDEA

### Installation

1. **Clone the repository**

   ```sh
   git clone https://github.com/Fahim-Mahtab/flutter_task_manager.git
   cd flutter_task_manager
   ```

2. **Install dependencies**

   ```sh
   flutter pub get
   ```

3. **Run the app**

   - For Android/iOS:
     ```sh
     flutter run
     ```
   - For Web:
     ```sh
     flutter run -d chrome
     ```

## Usage

- Tap the "+" button to add a new task.
- Tap on a task to view or edit its details.
- Swipe left/right (or use long press) to delete or mark a task as completed.
- Use categories for better task organization.
